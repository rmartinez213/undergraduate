- Problem 1 - Lucky Lively Llama -

Lucky lively llama, is a program that has a randomly looped value and will display the number and 
'Lucky' if the number is divisible by 7. The program will display the value and 'Lively' if the number
is divisible by 11. If the number is divisible by 13, print the number and  “llama.” I If the number
isn’t divisible by any of these, or is divisible by two of them, print nothing. If the number is divisible
by all three, print the number and  “What a llama!”
